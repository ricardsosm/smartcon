  • Smartcon

    Para o uso do Smartcon é necessário alguns passos, para que o contrato que você deseja possa satisfazer as informações mínimas nescessárias para a crição dos Smart Contracts

  • Para o uso do Smartcon a primeira coisa a ser feito é cadastrar um Cliente, clicando no Menu Clientes, aparecerá um novo menu lateral, onde voce pode clicar em Novo Cliente

  • Após o cadastro de um Cliente é preciso vincular uma Carteira para esse Cliente

    Isso pode ser feito clicando no menu esquerdo em Gerar Carteira, onde aparecerão as chaves públicas e privadas.

    OBS: caso queira usar uma Carteira já existente você pode simplesmente substituir as chaves públicas e privadas que aparecem na tela

    Você pode por saldo, para testes, em sua conta no site https://faucet.ropsten.be
